# Pixel Perfect Team Bug Smashers - Submission

## How to use
### 1\. Clone this Repository

```bash
git clone https://github.com/shreyanshTechz/nitjconf.git
```

### \. Navigate to the directory

```
cd nextly-template
```

### . Install dependencies

```
yarn install
# or
npm install
```

### . Run Dev Server

```
yarn dev
# or
npm run dev
```
